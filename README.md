# Algoritmo-de-Compresion-LZW
Implementación del algoritmo de compresión LZW para comprimir archivos y directorios.

Para compilar ejecutar el comando:
 ```
 g++ lzw.cpp -o lzw
 ```
Para comprimir:
 ```
 ./lzw [files] [files] > [name].lzw 
 ```
Para descomprimir:
 ```
 ./lzw -u [name].lzw 
 ```
Para help:
  ```
  ./lzw -h
  ```
